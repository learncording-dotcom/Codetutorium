# Phone Mirror Tool

A comprehensive phone mirroring and remote session management tool for ethical hacking and security testing.

## Features

- 🔗 **Session-based Mirroring**: Create unique sessions via shareable links
- 📱 **Multi-Platform Support**: Android and iOS compatibility
- 🎥 **Real-Time Screen Capture**: Live device screen streaming
- 🖱️ **Remote Control**: Control device inputs remotely
- 📹 **Session Recording**: Record all mirrored sessions
- 🔐 **Secure Authentication**: Token-based session management
- 🌐 **Web Dashboard**: Browser-based monitoring interface
- 📊 **Session Analytics**: Track session history and metrics

## Disclaimer

⚠️ **Important**: This tool should only be used for:
- Educational purposes
- Authorized security testing and penetration testing
- Testing on your own devices
- With explicit written consent from device owners

Unauthorized access to devices is illegal. Use responsibly.

## Quick Start

### Prerequisites

- Node.js 16+
- Python 3.8+
- Android SDK (for Android testing)
- iOS SDK (for iOS testing)

### Installation

1. Clone the repository
2. Install dependencies for each component
3. Configure environment variables
4. Start the server
5. Deploy client to test devices
